特质trait

1、

trait Logger{
  def logger{
    println("Logger")
  }
}

trait ConsoleLogger extends Logger{
  override def logger{ //因为Logger已经实现了，所以要加override，否则不用加
    println("ConsoleLogger")
  }
}

class MyTest extends Logger{
}

object Test extends App{
  val c = new MyTest with ConsoleLogger  //物质混入对象
  c.logger  //调用ConsoleLogger
  val x = new MyTest with Logger 
  x.logger  //调用ConsoleLogger，不会用Logger的，这里应该也会用子特质的
  val d = new MyTest   
  d.logger   //调用Logger
}

如果类混入了某个特质X，有方法A，则对象混入另一个拥有方法A的特质Y时会报错，因为是方法冲突，除非X和Y有继承关系，或者X ext Y，或者Y ext X

trait Logger{
  def logger{
    println("Logger")
  }
}

trait ConsoleLogger{
  def logger{
    println("ConsoleLogger")
  }
}

class MyTest extends Logger{
}

object Test extends App{
  val c = new MyTest with ConsoleLogger
  c.logger
}

报错：

//方法冲突
anonymous class $anon inherits conflicting members:
method logger in trait Logger of type => Unit  and
method logger in trait ConsoleLogger of type => Unit
(Note: this can be resolved by declaring an override in anonymous class $anon.)
val c = new MyTest with ConsoleLogger
          ^

如果Logger和ConsoleLogger的方法不一样，就不会有这个问题
